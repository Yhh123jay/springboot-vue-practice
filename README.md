1、用ref封装的数据需要用value取到

```js

```

2、mapper.xml文件扫描配置

```yaml
mybatis:
  mapper-locations: classpath:mapper/*.xml #扫描所有mybatis的所有xml文件
  configuration:
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
```
